# Logo "Servir+" - Fichiers à intégrer

## 📁 Fichiers inclus

- icon.svg - Logo principal
- icon-simple.svg - Logo simplifié
- banner.svg - Banner horizontal
- icon-*.png - 8 icônes PNG (déjà générées)
- logo-preview.html - Page de visualisation

## 🔧 Installation

Copiez tous ces fichiers dans le dossier public/ de votre projet :

```bash
# Extraire l'archive
unzip logo-servir-plus.zip

# Copier dans votre projet
cp logo-servir-plus/* /chemin/vers/votre/projet/public/
```

## ✅ C'est tout !

Les icônes sont déjà générées, pas besoin de les recréer.

Pour visualiser : ouvrez public/logo-preview.html dans votre navigateur.
